## 1 资料内容介绍

资料分为课程要求以及知识整理两部分。

- 课件资源来自于课程组提供的资料；
- 课程实验来自于华为云和课题组联合提供的资料；
- 结课项目为作者本人的，有一定参考价值。

## 2 课程建议

- 课程形式为理论课+日常实验+结课项目（pre可加分）；
  - 理论课主要听懂逻辑，对云有基本的概念，形成逻辑框架，后面的实验会思路清晰，事半功倍；
  - 日常实验建议先看懂要做什么，再按照实验指导动手去做，注意要总结过程中遇到了什么困难以及相应的解决办法，以此验收。
  - 结课项目需要自己搭建一个完整的项目，涵盖了日常实验内容，届时老师会发布具体的要求，根据要求和自己的想法做就好，但是要注意华为云代金券的合理使用（做的太拖拉可能要自掏腰包👀），以及要找到自己设计的优点以便pre。个人建议高度容器化会比较清晰、方便~

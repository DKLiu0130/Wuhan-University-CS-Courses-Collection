## 1 资料内容介绍

资料包括项目要求、项目代码和项目报告三部分。

- 项目代码实现了一个教学数据库系统，涉及存储管理、查询执行、索引、事务、并发控制和故障恢复等模块；
- 项目报告记录了数据库系统实现过程，可结合代码理解各阶段任务。

## 2 课程建议

- 课程以项目实现为主，建议按任务顺序推进，每完成一个模块及时运行测试并保留结果；
- 阅读代码时先理清各模块接口和数据流，再定位具体实现，避免只针对测试结果修改；
- 报告应如实记录设计思路、实现细节和测试情况，方便后续检查与总结。

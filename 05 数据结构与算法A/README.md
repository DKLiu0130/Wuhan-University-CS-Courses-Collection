- 课件资料为2024年度数据结构A课程组的幻灯片；
- 知识点整理是非常全面、细致的，由于我当时没有学习过C++，只学习过C语言，对模版类等不太熟练，有部分这方面的整理在里面；
  - 数据结构期末考试编程部分总结：涉及到可能的代码题目，这是和当年的考情有关系的，大家要根据老师提供的重点和模拟题进行调整；

  - STL常见用法总结：这是C++课程中应该掌握的一部分内容。


## 2 备考建议

- 期末考试为闭卷机考，难度和不确定性还是比较大的；
- 对于前面的概念题，一定要做一做考研题（或者这几年机考的题目，如果你能找到），因为这部分丢分很可惜，而且确定性比较强；
- 对于后面的编程题，要熟练所有的基本结构，比如链表、字符串等等。
  - 一个比较好的训练方法就是STL有什么功能就自己复现什么功能；

  - 还可以考前早点到机房，把这些结构先敲出来，以备考试的时候使用。

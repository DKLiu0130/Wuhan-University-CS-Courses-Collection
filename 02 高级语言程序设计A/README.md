## 2 备考建议

- 期末考试为开卷机考，选择主要是基础概念，难度比较大；
  - 比如，数组指针和指针数组、虚函数、虚类等等。

- 填空和编程题，有一部分考察基础语法，还有一部分可能涉及算法；
  - 这部分重要的是搞清楚输入输出方法，比如用“Ctrl+z”结束输入、`sstream`等等。

- 往年题价值不大，关注模拟编程题；

## 🚀 使用建议

为了更高效地利用本仓库，建议按以下步骤进行：

1. **学期前准备**：在新学期开始前，下载对应课程的文件夹，初步了解课程大纲与知识结构。
2. **先读本课程 README**：每个课程文件夹内均有独立的 `README.md`，包含学习建议、重点梳理与参考资料说明，请先通读。
3. **结合堂学习**：资料仅为辅助，请务必以课堂讲授和最新官方课件为核心，动态更新自己的知识体系。
4. **主动整理与提问**：鼓励在学习过程中自己整理笔记，如有疑惑可邮件交流，但更建议先与课程教师或同学讨论。

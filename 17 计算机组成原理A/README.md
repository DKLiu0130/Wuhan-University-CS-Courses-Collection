## 1 资料内容介绍

- 课件资料为2025年度计组课程组的幻灯片；
- 华科教材整理：他们采用的是MIPS系统，但是其他内容和我们类似，对应的教材质量很高，个人认为比国外的教材强，可以更高效；
- 知识点整理是非常全面、细致的，这个整理是我多次向班级和年级同学讲解、反复修改的成果，很有参考价值；
  - 知识点整理：覆盖所有的考试内容，理解是比较深入的，如果有看不懂的地方，可以反复理解，不要惊慌；
  - 期末习题集：内含我自己写的题目答案，网上的答案至少有一半都是错的；
  - 冒险相关整理：这是我的一个讲座，针对冒险这个板块进行了详细的梳理，属于高分必备。

- 课程实验是比较容易的verilog编程，内含基础代码和实验书，仅供参考。

## 2 备考建议

- 期末考试为闭卷考试，难度比较大，最近改为机试，但是基本还是之前的题型，不必过于担心；
- 一定要侧重于理解，考试题型变化多端，只能保证基本上是每个板块一个题目；
  - 多级流水线是重难点，一定要区分几种冒险；

  - 高速缓存是重难点，一定要搞清楚流程和架构。
  
- 往年试卷很有参考价值。
